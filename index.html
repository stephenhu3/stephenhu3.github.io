Autonomous Electromagnetic Robot

Solenoid receivers detect voltages based on distance from remote’s electromagnetic transmitter, which enables the robot to follow the remote’s physical location.

Responds to parallel park, reverse orientation, and adjust following distance commands.

Embedded software written in C with embedded assembly flashed to a modified Atmel AT89 microcontroller.

Autonomous control can be manually overridden by plugging in and using the modified arcade joystick.

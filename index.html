Open Source Mafia Hobby Project

Open source Java-based platform for the Mafia murder mystery party game currently available on desktop and Android

  • Each player is randomly assigned a role and an alignment towards the Mafia or the Town
  • Every night, each player performs his/her role's ability and votes towards eliminating a suspicious player
  • This is a game of deceit, persuasion, teamwork, and careful planning
  • The goal of each player depends on the winning condition of the respective roles
  • Game rules and roles are based on the SC2 Mafia Wiki: http://wiki.sc2mafia.com/index.php?title=Main_Page

Autonomous Electromagnetic Robot

Solenoid receivers read voltage based on distance from remote's electromagnetic transmitter and follows the remote's physical location.

The remote can send commands to parallel park, reverse orientation, and adjust following distance.

Robot's embedded software was written in C with some embedded assembly flashed to a modified Atmel AT89 microcontroller.

Autonomous control can be manually overridden by plugging in and using the modified arcade joystick.

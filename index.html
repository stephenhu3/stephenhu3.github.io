Open Source Mafia Hobby Project

Open source Java-based platform for the Mafia murder mystery party game currently available on desktop and Android 4.0+.

  • Each player is randomly assigned a role and an alignment towards the Mafia or the Town.
  • Every night, each player performs his/her role's ability and votes towards eliminating a suspicious player.
  • This is a game of deceit, persuasion, teamwork, and careful planning.
  • The goal of each player depends on the winning condition of the respective roles.
  • Game rules and roles are based on the SC2 Mafia Wiki: http://wiki.sc2mafia.com/index.php?title=Main_Page

Autonomous Electromagnetic Following Robot

  • Robot's solenoid receivers read voltage based on distance from remote's electromagnetic transmitter and follows the remote's movements.
  • Robot's embedded software was written in C with some embedded assembly flashed to a modified Atmel AT89 microcontroller. Can send commands from the remote.
  • Robot can also be manually controlled by plugging in and using the modified arcade joystick.
    • My role included:
  • designing and coding the autonomous following algorithm
  • designing and coding the robot's rotation and forwards/backwards movement
  • implementing robot functionality of performing parallel parking, rotate 180 and reverse orientation
  • implementing robot functionality of receiving and executing commands from the remote (move closer, move further, parallel park, etc.)
  • troubleshooting and debugging the robot's embedded software
  • regression testing and data collection of robot's quality of functionality (max range from transmitter, accuracy of distance reading, etc.)
  • writing and editing a technical report on the problem specification, design process, and solution generation
